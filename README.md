# hello-processing
Proyectos de Processing para ESO y Bachillerato. Actividades listas para clase con código y explicaciónes

Este repositorio forma parte de una colección de materiales para materias como:
- Programación y Computación
- Pensamiento Computacional  
- TIC  
- Computación y Robótica 

## Objetivos del repositorio
- Proporcionar proyectos y actividades listos para usar en clase.
- Explicar conceptos clave de Processing de forma clara y progresiva.
- Facilitar ejercicios motivadores de animación e interacción visual.
- Ofrecer actividades ampliables según el nivel del alumnado.
- Servir como repositorio público y gratuito para profes de informática.

## Estructura del repositorio

Cada proyecto incluye:
- Código en formato `.pde`
- Comentarios con:
  - Objetivo
  - Conceptos trabajados
  - Explicación del código
  - Propuestas de ampliación
